# FastAPI Learning Repository

Welcome to my **FastAPI Learning Repository**! This repository documents my journey of learning and mastering **FastAPI**, one of the fastest and most modern Python frameworks for building high-performance web APIs. My goal is to gain a strong understanding of backend development by building real-world projects and implementing industry-standard best practices.

Day - 1
    - Understand the working of FastAPI
    - learn about GET API
    - learn about asynchronous function that handles requests to this endpoint.
    - test API on web and also automatically generated Swagger UI.
    
Days - 2 
    - Learn about Path Parameters
    - learn how to define path parameters
    - learn how to access path parameters
    - learn about routers
    
Days - 3
    - learn about Query Parameters
    - learn how to define query parameters
    - learn how to access query parameters

Days - 4 
    - learn about Request Body with Pydantic
    
Days - 5
    - Pydantic Field Validation
    - create custom validation messages for Pydantic
